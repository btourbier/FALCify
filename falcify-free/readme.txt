FALCify Free ajoute un bouton "Lire en version facile" pour basculer instantanément vers une version FALC du contenu (si elle est fournie dans la métabox).

== Description ==
- Version MVP : pas d’appel API; vous collez la version FALC dans la métabox de l’éditeur.
- Bascule instantanée sans rechargement de page.
- Option pour activer/désactiver le bouton globalement.
- Stockage séparé dans post_meta: _falcify_falc
- i18n prêt (text domain: falcify-free).

== Installation ==
1. Téléversez le dossier `falcify-free` dans `/wp-content/plugins/` ou installez via WP Pusher depuis GitHub.
2. Activez le plugin.
3. Réglages > FALCify Free : cochez/décochez l’affichage du bouton.
4. Éditez votre article : renseignez la version FALC dans la métabox FALCify.

== Frequently Asked Questions ==
= Le bouton n’apparaît pas ? =
- Vérifiez que l’option “Activer le bouton” est cochée.
- Assurez-vous qu’une version FALC existe dans la métabox du contenu.
